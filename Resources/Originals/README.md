# Originals — Where to Find Source Material

Canonical URLs for the source material this framework cites. Nothing is mirrored locally — the originals are authoritative and live on the open web.

For copyrighted material (books, blog posts, papers) see [`Citations/`](Citations/) for ISBNs, DOIs, and purchase links.

## Why we don't mirror

A web page being publicly readable is **not** the same as being licensed for redistribution, and even when redistribution is permitted (as with all of the above), mirrored copies go stale. OWASP Top 10 has both 2017 and 2021 editions; ASVS has 4.x and a draft 5.x; 12factor.net occasionally edits in place. Linking to canonical URLs keeps citations honest.

For copyrighted material we cannot redistribute (books, Fowler's bliki, Joel on Software, academic papers) see [`Citations/`](Citations/) for purchase / library / DOI references.
